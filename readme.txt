En este trabajo práctico desarrollamos un algoritmo de inteligencia artificial capaz de reconocer y clasificar con el mayor nivel de precisión posible imágenes de diferentes camisetas de equipos de fútbol argentino, asociándolas con el club correspondiente. 

Modelo de RNA utilizado 
En la resolución de este proyecto, comenzamos por definir un modelo de aprendizaje supervisado, unidireccional, con un algoritmo de entrenamiento por retropropagación del error (back propagation).
Este modelo ajusta los pesos de las conexiones entre las neuronas de la red en función de los errores de predicción, mejorando la precisión del resultado.
Las Redes Neuronales Convolucionales (CNN) generalmente son más adecuadas para tareas de clasificación de imágenes debido a su capacidad para capturar características espaciales y patrones locales. Sin embargo, nos decidimos por el modelo mencionado por razones de simplicidad, y para evaluar el funcionamiento y la precisión de este tipo de red.
